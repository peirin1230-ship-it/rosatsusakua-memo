# ロサTス作ア メモ

高松智史氏の著書で紹介されている仕事術フレームワーク **「ロ→サ→T→ス→作→ア」** を、日常業務でサッと使えるメモツールとして実装した React コンポーネントです。

## フレームワークとは？

「ロサTス作ア」は、仕事を正しい順序で進めるための6ステップです。

| ステップ | 名称 | やること |
|----------|------|----------|
| **ロ** | 論点 | そもそも何を答えるべきか？を「〜すべきか？」の形で定義する |
| **サ** | サブ論点 | 論点を解くための「問い」に分解する（3〜5個） |
| **T** | TASK | 各サブ論点に答えるための具体的な作業を設計する |
| **ス** | スケジュール | TASKの優先度・順番・期限を決めてWBSに落とす |
| **作** | 作業 | 手を動かす。ただし常に論点に立ち返る |
| **ア** | アウトプット | 最終成果物を仕上げる → そのあと必ずD（議論）へ |

**核心：** いきなりTASKに飛ばない。「ロ→サ」= 論点とサブ論点を先に立てるだけで、仕事の質が激変する。

## 機能

- 6ステップをタップで展開し、各ステップにメモを記入
- プロジェクト名の入力
- 進捗バーで記入状況を可視化
- メモをテキストファイルとしてエクスポート
- セルフチェックリスト付き
- 「TASKバカ注意報」で論点を飛ばしていないかリマインド
- ダークテーマUI

## 使い方

### 前提条件

- Node.js (v18以上推奨)
- React プロジェクト環境

### インストール

1. このリポジトリをクローンします：

```bash
git clone https://github.com/peirin1230-ship-it/rosatsusakua-memo.git
```

2. お使いの React プロジェクトに `rosatsusakua-memo.jsx` をコピーし、コンポーネントとしてインポートします：

```jsx
import RoSaTSuSaA from "./rosatsusakua-memo";

function App() {
  return <RoSaTSuSaA />;
}
```
