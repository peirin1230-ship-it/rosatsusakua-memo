# ロサTス作ア メモ

高松智史氏の仕事術フレームワーク「ロ→サ→T→ス→作→ア」を、日常業務でサッと使えるメモツールとして実装した Web アプリです。

**https://peirin1230-ship-it.github.io/rosatsusakua-memo/**

## フレームワーク

| ステップ | 名称 | やること |
|----------|------|----------|
| ロ | 論点 | そもそも何を答えるべきか？を「〜すべきか？」の形で定義する |
| サ | サブ論点 | 論点を解くための「問い」に分解する（3〜5個） |
| T | TASK | 各サブ論点に答えるための具体的な作業を設計する |
| ス | スケジュール | TASKの優先度・順番・期限を決めてWBSに落とす |
| 作 | 作業 | 手を動かす。ただし常に論点に立ち返る |
| ア | アウトプット | 最終成果物を仕上げる → そのあと必ずD（議論）へ |

いきなりTASKに飛ばない。「ロ→サ」を先に立てるだけで、仕事の質が激変する。

## 機能

### 構造化入力

サブ論点を個別の入力欄に分離し、各サブ論点に対して複数のTASKとスケジュールを紐づけられます。

```
ロ: 新規事業Xに参入すべきか？
├── サブ論点1: 市場規模は十分か？
│   ├── TASK: 業界レポート分析       Day1-2
│   └── TASK: ヒアリング3件実施       Day2-3
├── サブ論点2: 自社の強みを活かせるか？
│   └── TASK: 競合ベンチマーク        Day3-4
└── サブ論点3: 収益性は確保できるか？
    └── TASK: 事業計画シミュレーション Day4-5
```

### 表示モード

- 個別 — ロ → サ(+T+ス) → 作 → ア を順にカード展開して記入
- 一覧 — 全サブ論点とTASKを俯瞰しながら編集
- ツリー — 論点→サブ論点→TASKの分解構造を可視化

### クラウド同期

- Google アカウントでログインすると、メモが自動的にクラウドに保存される
- どの端末・ブラウザからでも同じデータにアクセス可能
- ログインしない場合はブラウザのローカルストレージに保存（従来動作）

### その他

- 複数プロジェクト管理（テンプレートから新規作成可能）
- テキスト / Markdown でエクスポート・インポート
- AI壁打ち（プロンプトをコピーして claude.ai に貼り付け）
- ダーク / ライトモード切替
- セルフチェックリスト

## 使い方

ブラウザで開くだけで使えます。インストール不要。

```
https://peirin1230-ship-it.github.io/rosatsusakua-memo/
```

ローカルで使う場合：

```bash
git clone https://github.com/peirin1230-ship-it/rosatsusakua-memo.git
# index.html をブラウザで開く
```

## クラウド同期のセットアップ

ログイン機能を有効にするには Firebase の設定が必要です。手順は [SETUP.md](SETUP.md) を参照してください。
