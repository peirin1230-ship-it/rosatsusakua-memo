# ロサTス作ア メモ

高松智史氏の著書で紹介されている仕事術フレームワーク **「ロ→サ→T→ス→作→ア」** を、日常業務でサッと使えるメモツールとして実装した Web アプリです。

**今すぐ使う → [https://peirin1230-ship-it.github.io/rosatsusakua-memo/](https://peirin1230-ship-it.github.io/rosatsusakua-memo/)**

## フレームワークとは？

「ロサTス作ア」は、仕事を正しい順序で進めるための6ステップです。

| ステップ | 名称 | やること |
|----------|------|----------|
| **ロ** | 論点 | そもそも何を答えるべきか？を「〜すべきか？」の形で定義する |
| **サ** | サブ論点 | 論点を解くための「問い」に分解する（3〜5個） |
| **T** | TASK | 各サブ論点に答えるための具体的な作業を設計する |
| **ス** | スケジュール | TASKの優先度・順番・期限を決めてWBSに落とす |
| **作** | 作業 | 手を動かす。ただし常に論点に立ち返る |
| **ア** | アウトプット | 最終成果物を仕上げる → そのあと必ずD（議論）へ |

**核心：** いきなりTASKに飛ばない。「ロ→サ」= 論点とサブ論点を先に立てるだけで、仕事の質が激変する。

## 機能

### サブ論点ベースの構造化入力

このツールの最大の特徴は、**サブ論点を個別の入力欄に分離**し、各サブ論点に対して**複数のTASKとスケジュールを紐づけられる**点です。

```
ロ: 新規事業Xに参入すべきか？
├── サブ論点1: 市場規模は十分か？
│   ├── TASK: 業界レポート分析       📅 Day1-2
│   └── TASK: ヒアリング3件実施       📅 Day2-3
├── サブ論点2: 自社の強みを活かせるか？
│   └── TASK: 競合ベンチマーク        📅 Day3-4
└── サブ論点3: 収益性は確保できるか？
    └── TASK: 事業計画シミュレーション 📅 Day4-5
```

- サブ論点を何個でも追加・削除できる
- 各サブ論点に対してTASKを複数追加でき、それぞれにスケジュールを設定
- TASKが宙ぶらりんにならず、必ずサブ論点に紐づく

### 表示モード

- **個別モード** — ロ → サ(+T+ス) → 作 → ア を順にカード展開して記入
- **一覧モード** — 全サブ論点とTASKを俯瞰しながら編集
- **ツリーモード** — 論点→サブ論点→TASKの分解構造をツリー図で可視化

### データ管理

- **自動保存** — 入力するたびにブラウザに自動保存。リロードしてもデータが消えない
- **複数プロジェクト管理** — プロジェクトの新規作成・切替・削除が可能
- **テンプレート** — 「新規事業検討」「業務改善」「問題解決」等のテンプレートからプロジェクトを開始

### エクスポート・インポート

- **テキスト (.txt)** / **Markdown (.md)** でエクスポート
- エクスポートしたファイルをインポートして復元

### AI壁打ち

- 各ステップで「🤖 AIに相談」ボタンを押すと、現在のメモ内容を含むプロンプトがクリップボードにコピーされる
- そのまま claude.ai に貼り付けて、論点やサブ論点を一緒に磨ける

### その他

- **ダーク/ライトモード切替** — ヘッダーの☀️/🌙ボタンで切替、選択は保存される
- **進捗バー** — ロ・サ・T・ス・作・アの記入状況を可視化
- **インタラクティブチェックリスト** — セルフチェック項目をクリックでON/OFF
- **TASKバカ注意報** — 論点を飛ばしていないかリマインド
- **タイムスタンプ** — 最終更新日時を表示

## 使い方

### ブラウザでそのまま使う（推奨）

以下のURLにアクセスするだけで使えます：

**https://peirin1230-ship-it.github.io/rosatsusakua-memo/**

インストール不要。スマホのホーム画面に追加すればアプリのように使えます。

### ローカルで使う

`index.html` をダウンロードしてブラウザで開くだけで動作します。

```bash
git clone https://github.com/peirin1230-ship-it/rosatsusakua-memo.git
```
